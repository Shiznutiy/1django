# First djamgo project

## Install
    python -m venv venv
    cd .\venv\Scripts\activate
    pip install django
    pip freeze > requirments.txt
    django-admin startproject <имя проекта>
## Run
    перейти в директорию проекта
    python .\manage.py runserver
    перейти по адрерсу http://127.0.0.1:8000/
